CSS Functions:

Mathematical Functions:

calc(): Performs calculations to determine a value, e.g., width: calc(100% - 20px); min(), max(), clamp(): Used for responsive sizing by setting minimum, maximum, or ideal values within a range, e.g., font-size: clamp(1rem, 5vw, 2rem);

Color Functions:

rgb(), rgba(), hsl(), hsla(): Define colors using different color models, with a representing the alpha (opacity) channel, e.g., background-color: rgba(255, 0, 0, 0.5); color-mix(): Blends two colors together, e.g., background-color: color-mix(in srgb, blue 50%, red);

Gradient Functions:

linear-gradient(): Creates a linear color transition, e.g., background: linear-gradient(to right, red, yellow); radial-gradient(): Creates a radial color transition.
